# STM32 Ramp Waveform Generator

## Overview

This project implements a Ramp Waveform Generator using an STM32 microcontroller and Embedded C.

The project was developed during a one-month Embedded Systems Internship at C.V. Raman Global University, Bhubaneswar (July 2024). It demonstrates practical firmware development, peripheral configuration, timer management, and hardware-software integration using STM32CubeIDE.

---

## Project Highlights

* Developed firmware using Embedded C
* Configured STM32 peripherals and GPIOs
* Implemented ramp waveform generation logic
* Utilized STM32 HAL libraries
* Performed hardware testing and debugging
* Structured project using STM32CubeIDE

## Working Principle

The STM32 microcontroller is configured through STM32CubeIDE and HAL drivers. The firmware initializes required peripherals and executes ramp waveform generation logic. The generated output waveform can be monitored and verified using hardware testing equipment.
